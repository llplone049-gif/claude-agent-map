1

agentmap フォルダの中身を .claude に移動する

永峰からもらった agentmap フォルダを開くと、中に2つのフォルダが入っています。

agentmap/（永峰からもらったフォルダ）
    ├── commands/
    └── agent-map/

この2つのフォルダを、自分のプロジェクトの .claude/ フォルダの中にコピーします。

自分のプロジェクトフォルダ/
    └── .claude/
            ├── commands/  ← ここにコピー
            └── agent-map/  ← ここにコピー
Finderで .claude フォルダが見えない場合は Cmd + Shift + . で隠しファイルを表示できます。
.claude フォルダがまだない場合は、新規フォルダを作って名前を .claude にしてください。
2

Claude Code でセットアップを実行する

自分のプロジェクトで Claude Code を開いて、セットアップ用プロンプト.md の中身をそのままコピーして貼り付けるだけです。

  • 名前の設定（初回のみ）
  • Notion API キーの設定（Claude Code が聞いてくれます）
  • 動作確認・チームページへの初回投稿
Claude Code がすべての手順をエスコートしてくれます。指示に従って答えるだけで完了します。
3

次回から

Claude Code を開いたら、以下のコマンドを入力するだけで更新できます：

/agent-map
  • エージェント名・役割・使用ツール・トークン数のみ共有されます
  • プロンプトの内容・MDファイルの中身は一切送信されません
  • 約5分後にチームページに反映されます

よくある質問

どのくらいの頻度で実行すればいい？
Claude Code を開くたびに実行するのがベストです。トークン数はその時点のスナップショットなので、まめに更新するほどチームの状況がリアルに反映されます。
毎回忘れずに実行するには？
Claude Code の設定ファイルにフック（自動実行）を追加するのがおすすめです。
プロジェクトの .claude/settings.json に以下を追加すると、セッション開始時に自動で /agent-map を実行するよう Claude Code に指示できます：
{ "hooks": { "SessionStart": [ { "matcher": "", "hooks": [ { "type": "command", "command": "echo 'Agent Map の更新タイムです！/agent-map を実行してください'" } ] } ] } }
設定すると、セッション開始時にリマインダーが自動で表示されます。
※ フック設定がわからなければ永峰に聞いてください。
エージェントを使っていなくても参加できる？
もちろんです！エージェント0体でも投稿できます。
うまくいかない場合は？
永峰に声をかけてください。一緒にセットアップします。
